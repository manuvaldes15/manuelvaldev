# ValDEV

Mi Portafolio personal Manuel Valdés como software developer. Es un sitio multipágina, pensado para mostrar mis ideas y proyectos, espero les guste :D


## Vista general

Cada pantalla toma una idea visual intentando siempre ser interactiva:

| Página | Propuesta visual |
|---|---|
| `index.html` | Gafete suspendido con física de resorte y tarjetas flotantes con parallax |
| `proyectos.html` | Carpetas apiladas con pestañas que se levantan al interactuar |
| `stack.html` | Nube de habilidades en forma de pills de color |
| `aptitudes.html` | Rompecabezas 2x2 que se arma al hacer scroll |
| `certificaciones.html` | Recorrido serpenteante SVG con formación, experiencia y voluntariado |
| `redes.html` | Avatar animado con contactos presentados como tickets |

## Características

- Diseño responsive para escritorio y móvil.
- Navegación tipo dock con estética macOS.
- Soporte de idioma español e inglés.
- Modo claro y oscuro.
- Iconografía SVG.
- Persistencia de preferencias en `localStorage`.

## Tecnologías

- HTML5
- CSS3
- JavaScript vanilla
- SVG para ilustraciones y animaciones

## Personalización rápida

| Elemento | Archivo |
|---|---|

| Traducciones ES/EN | `js/i18n.js` |

| Contactos y redes | `redes.html` |
